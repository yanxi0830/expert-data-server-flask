What is Neural Data Server?


Neural Data Server is a search engine that uses A.I. to find the best data to pre-train your deep neural models for your target application. In the new era of ever increasing number of massive datasets, selecting the relevant pre-training data itself is a critical issue. To address this, we introduce Neural Data Server, a web search engine that uses A.I. to find the best data to pre-train your deep neural models for your target application. Neural Data Server indexes several popular Machine Learning datasets in its quest to serve optimal data to ML users, and returns IDs of relevant data for your application.

Watch the short video below on how Neural Data Server works. Also checkout our guide on how to use our platform. For further technical information about NDS please see our paper.
